# Dog Dash

A browser-based 3D neon runner game starring a custom dog model.

## Play

Open `index.html` through a local server or GitHub Pages. The game entry is:

```text
3d-runner-rich.html
```

## Included

- Three.js runtime files are vendored under `vendor/three/`.
- Dog FBX animations and textures are under `dog-assets/`.
